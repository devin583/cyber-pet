# CyberPet 🐾

一只活泼灵动的桌面宠物,主打 Windows,支持「素材包」方式自由更换形象,无需改代码。

![](assets/icon.png)

## 功能一览

**互动**
- 🖱️ 单击戳它:会回头搭话;2 秒内连戳 4 次会被戳晕、生气
- 🖱️ 双击:开心地蹦一下
- ✋ 拖拽:可以拎起来到处放,甩出去会真的飞出去——碰墙反弹、落地压扁,摔疼了还会抱怨
- 🤚 抚摸:在它身上来回滑动鼠标,会眯眼冒爱心
- 🌀 滚轮搓它:转晕
- 🍙 喂食:右键菜单喂饭团
- 💤 睡觉:电脑空闲 5 分钟自动入睡,点它即醒;也可手动哄睡
- 💬 打字聊天:右键 →「和它说话…」,关键词回复;还能用话指挥它(说"睡觉""跟我走""讲个笑话""石头"猜拳……)
- 🤖 AI 聊天(可选):设置里填任意 OpenAI 兼容接口(OpenAI / DeepSeek / 本地 Ollama)即可自由对话,带素材包人设;不填则用免费本地规则引擎
- 🚶 跟我走:开启后它会一路小跑跟着你的鼠标
- 🎮 小游戏:猜拳(记胜负)、抛硬币、掷骰子、今日运势(宜/忌,按日期固定)

**联网趣味内容**(断网自动降级为本地台词)
- ✨ 一言:随机文学/动漫/诗词短句,闲聊时也会偶尔冒出来
- ⛅ 实时天气:基于 IP 定位,免配置

**自主行为(活泼的部分)**
- 随机眨眼、伸懒腰、沿任务栏散步(偶尔跑一大段)
- 随机闲聊、按时段问候(早/午/晚/深夜)
- 空闲时眼神会跟着你的鼠标转向

**通知与提醒**
- ⏰ 喝水提醒 / 久坐提醒:宠物会抱住工作窗口并把它推开,露出休息提示;可完成、延后 10 分钟或跳过
- 🫧 提醒动效可在设置中关闭或即时预览;关闭后仍保留宠物气泡 + 系统通知
- 🕐 整点报时(默认关)
- 🍅 番茄钟:25 分钟专注,期间它安静陪伴,结束时系统通知
- 提醒台词可由素材包自定义

**系统集成**
- 常驻系统托盘:显示/隐藏、切换素材包、调整大小、开关提醒、开机自启
- 置顶但不抢焦点,鼠标不在宠物身上时完全点击穿透,不影响正常工作

## 运行

```bash
npm install
npm start
```

## 下载安装包

在 [Releases](https://github.com/devin583/cyber-pet/releases) 页面下载:Windows 用 `.exe`(NSIS 安装器),macOS 用 `.dmg`(未签名,首次打开需右键 → 打开)。

## 打包

本地打包:

```bash
npm run dist:win     # Windows 安装包(在 Windows 上执行)
npm run dist:mac     # macOS dmg
```

自动发布:推一个 `v*` 标签即触发 GitHub Actions 双平台打包并自动发 Release:

```bash
git tag v0.2.0 && git push origin v0.2.0
```

## 自定义形象(素材包)

内置形象是「团子猫」。想换成自己的角色,两种方式:

- **一键导入**:右键/托盘菜单 →「导入素材包…」,选 zip 或文件夹,导入成功自动换装
- **手动放置**:「打开素材包文件夹」→ 新建文件夹放入帧图片和 `pack.json` → 菜单里选中

完整规范见 [docs/asset-pack-spec.md](docs/asset-pack-spec.md)。最少只需要一张 `idle` 图就能跑。

## 设置文件(高级配置)

右键/托盘菜单 →「设置文件」直接打开 `settings.json`,**保存后自动生效**(无需重启):

```jsonc
{
  "weatherCity": "",        // 天气城市,如 "Beijing";留空按 IP 定位
  "waterIntervalMin": 60,   // 喝水提醒间隔(分钟)
  "sitIntervalMin": 45,     // 久坐提醒间隔(分钟)
  "reminderAnimation": true,// 宠物推开窗口提醒动效
  "localApi": true,         // 本地通知 API 开关
  "apiPort": 12580          // 本地 API 端口
}
```

## 本地通知 API(外部接入)

宠物在 `127.0.0.1:12580` 监听一个只限本机的 HTTP 接口,任何脚本/程序都能让它说话或发系统通知:

```bash
# 让宠物冒气泡说话
curl "http://127.0.0.1:12580/say?text=构建完成啦"

# 气泡 + Windows 系统通知
curl -X POST http://127.0.0.1:12580/notify \
     -d '{"title":"CI 结果","body":"全部测试通过 ✅"}'
```

用它可以接入任何来源的提醒:CI 完成、日程到点、邮件关键字、订阅更新……写一行 curl 就行。
定时任务、RSS 订阅轮询、CI 桥接等完整示例见 [docs/integrations.md](docs/integrations.md)。

## 项目结构

```
main.js                 # 主进程:窗口/托盘/素材包扫描/提醒调度/系统通知
preload.js              # IPC 桥
renderer/pet.js         # 宠物核心:状态机、物理(拖拽/抛掷/反弹)、互动、右键菜单
renderer/style.css      # 气泡与菜单样式
packs/default/          # 内置素材包「团子猫」(17 帧 SVG,可作为模板参考)
docs/asset-pack-spec.md # 素材包制作规范
scripts/gen_icons.py    # 生成托盘/应用图标
```

## 常见问题

- **Windows 上窗口不透明/闪烁**:少数显卡驱动下可尝试在 `main.js` 顶部加 `app.disableHardwareAcceleration()`。
- **通知没弹出**:Windows 设置 → 系统 → 通知,确认允许该应用通知;「专注助手」开启时通知会被静默。
- **素材包没出现在菜单**:确认文件夹里有 `pack.json` 且包含 `states.idle`。
